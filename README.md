# Sistema de Recomendações de Receitas para Pessoas com Restrições Alimentares

![logo](/src/assets/LogoSafeNutri.png)

Este é o repositório oficial do SafeNutri, um sistema de recomendações de receitas para pessoas com restrições alimentares, uma aplicação desenvolvida para auxiliar os usuários a não precisar mais ficar horas procurando receitas que sejam seguras para ele, pois o sistema já vai filtrar e sugerir receitas fornecidas por profissionais, se baseando na sua restrição alimentar.

## Funcionalidades

- Cadastros dos usuários;
- Cadastro dos nutricionistas;
- ...

## Tecnologias Utilizadas

- **Frontend:**
  - [Angular](https://angular.io/): Framework para construção de interfaces de usuário dinâmicas.
  - [Tailwind CSS](https://tailwindcss.com/): Framework CSS para estilização customizada e responsiva.

Pré-requisitos

- Node.js
- Angular CLI @21

## Instruções de Instalação

- Clone o repositório:

  - via SSH: git clone <git@github.com>:SafeNutri-IFTM/safenutri-front.git

- Instale as dependências:

  - npm install

- Execute o servidor de desenvolvimento:

  - ng serve

- Acesse a aplicação:

  - Abra o navegador e vá para <http://localhost:4200>.
